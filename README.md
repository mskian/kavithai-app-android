# 🔐 APK SHA256 Verification

Verify the APK checksum before installing to ensure the file is authentic and not modified.

### 📦 SHA256

```txt
sha256:8ce440e303bdd10fe4d99441d13b2cb5a8672cdc2f42724487e46e24f9e2177b
```

### 🛡️ Why Verify SHA256?

- ✅ Confirms APK integrity
- 🔒 Protects against tampered files
- 📥 Ensures safe installation
- 🧠 Confirms original GitHub release build

### 📱 Verify SHA256 on Android

Use apps like:

- ZArchiver
- Hash Checker
- MT Manager

### 💻 Verify SHA256 on Linux / macOS

```bash
sha256sum app-release.apk
```

Compare the generated hash with the SHA256 value above.
